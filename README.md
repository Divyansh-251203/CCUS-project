# 🌍 CO₂ Emissions Analysis for CCUS Insights

## 📌 Overview
This project analyzes global CO₂ emissions data to identify trends, high-emission regions, and insights relevant to Carbon Capture, Utilization, and Storage (CCUS).

## 🎯 Objectives
- Identify top CO₂ emitting countries
- Analyze global emission trends
- Study per capita emissions
- Provide insights for CCUS implementation

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Scikit-learn

## 📊 Key Features
- Data cleaning and preprocessing
- Visualization of emission trends
- Top emitter analysis
- Future emission prediction using Machine Learning

## 📈 Results
- Identified top CO₂ emitting countries globally
- Observed increasing emission trends over time
- Highlighted regions suitable for CCUS deployment

## 🧠 CCUS Insights
- High-emission countries are prime candidates for carbon capture
- Per capita emissions indicate inefficiencies in energy usage
- Predictive modeling helps in planning future CCUS strategies

## 📂 Dataset
Dataset used from:
https://github.com/owid/co2-data
